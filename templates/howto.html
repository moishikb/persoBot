Description

This chatBot developed in order to provide a platform for IT workers to assist them with common tasks during their daily work.
The system developed in Python and use the Flask module for managing the chet socket and enabling the web UI interface interaction

How to install

Please follow those steps in order to prepare your machine to run the bot:

1. install python - the system works with the latest 2.7.x version, you can download it from python site

2. install pip (in case your Python version doesnt include it already.

3. Deploy the Flask module by following the instructions in that site:

4. Deploy the Flask SOcketIO module by following the instructions in that site:

How to run

All you need to start using the bot is to:

  • run "python main.py" command from the root project folder
  • open that site index file url

How to use

1. Simply, you can ask any question you want. the bot will answer based on his knowledge and based on the operations that came OOTB

2. You can extend the bot knowledge by adding new questions and answers to the "bot_knowledge.json" file that exist in the dll folder

3. You can also extend the bot operations by creating new python script file, includes all functions you need and place it in the plugins folder

4. To use function, you can ask the bot chat "get list of actions" and you can run each one of them by entring the function name with @ as aprefix

5. To run bash or cmd command you can enter the command with # as a prefix
